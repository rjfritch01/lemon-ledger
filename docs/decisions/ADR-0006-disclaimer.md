# ADR-0006 — Mandatory Disclaimer on All Generated Forms

**Status**: Accepted (Stage 6, 2026-06-26)

## Context

See ADR-0004 (product reframing). Users must not mistake Lemon Ledger's output for a filed tax return or professional tax advice.

## Decision

Every page of every form PDF and every CSV output carries the following fixed disclaimer text:

> "INFORMATIONAL DRAFT — NOT FILED TAX ADVICE. Generated by Lemon Ledger for tax-preparation and record-keeping. Crypto-to-crypto trades are taxable disposals under US law. Review with a licensed tax professional before filing. Lemon Ledger and its author are not tax advisors."

Implementation:
- `pdf_base.py::disclaimer_paragraph()` returns a styled `Paragraph` with this text.
- Each PDF renderer (`pdf_8949.py`, `pdf_schedule_d.py`, `pdf_schedule_1.py`, `pdf_activity.py`) appends it as the last story element before `doc.build()`.
- `activity_report.py::to_csv()` includes it in the CSV file header comment row.

### Relationship to DRAFT watermark

The disclaimer is **separate** from the gate-held `[DRAFT — PENDING GATE RESOLUTION]` watermark on form titles. Both can appear simultaneously: the disclaimer is always present; the DRAFT tag is conditional on `is_draft=True`.

## Consequences

- Smoke tests verify that rendered PDFs contain the disclaimer bytes.
- Future form templates must call `disclaimer_paragraph()` before `doc.build()`.
- The disclaimer text is a constant in `pdf_base.py` (`_DISCLAIMER`); changing it requires updating all downstream tests that check for specific text.
